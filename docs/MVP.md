# Evalis MVP

## Problem

Schools need a simple web system for creating assessments, collecting student work, and managing results without relying on disconnected tools.

## Primary Users

- School admins
- Teachers
- Students

## Core MVP Features

### 1. Authentication and Roles

- Admin, teacher, and student roles
- Secure sign-in

### 2. Classes

- Create classes
- Enroll students
- Assign teachers

### 3. Assessments

- Create quizzes, assignments, and tests
- Define criteria and scoring rules

### 4. Submissions

- Students submit answers
- Teachers review submissions

### 5. Results and Reports

- Class-level performance overview
- Student-level progress summary

## Non-Goals for MVP

- Mobile app
- Desktop app
- Full LMS replacement
- AI-generated grading
- Advanced scheduling
- Parent portals

## First Milestone

Deliver a teacher workflow:

1. Sign in
2. Create class
3. Create assessment
4. Receive student submission
5. Review and score submission
6. Publish result
